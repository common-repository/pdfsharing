== Description ==

With this plugin you can add your direct downloads for pdf-documents in your wordpress-installation which supported
by pdfsharing.com. pdfsharing.com is a service so you can create direct downloads for social networks and wordpress.

Please lookup pdfsharing.com for more informations.

[Plugin Website](http://pdfsharing.com)

== Documentation ==

It is simple to add a direct pdf-download in your site. Add the widget into your area, add the sharing-link from
pdfsharing.com. If you want to use an other template for your site, you can change and modify the template in the widget. (look at the widget)

You can use the follow template-keys:

* &lt;!--LINK--&gt;
* &lt;!--IMAGE--&gt;
* &lt;!--TITLE--&gt;
* &lt;!--DESCRIPTION--&gt;

If you need a documentation please contact our [Support pdfsharing](support@pdfsharing.com)

== Installation ==

1. Upload the folder _/pdfsharing/_ to the _/wp-content/plugins/_ directory
2. Activate the plugin through the __Plugins__ menu in WordPress
3. Visit __Appearance > Widgets__ to add pdfsharing in the areas
4. If you have follow the steps, you can add you sharing-link from pdfsharing.com to the widget
